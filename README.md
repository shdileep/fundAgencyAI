# FundAgencyAI

A modern funding opportunity management platform that enables institutions to publish, manage, and track research funding calls with role-based access and real-time updates.

## Overview

FundAgencyAI is a web application that centralizes funding opportunities for students, researchers, and institutions. Administrators can publish funding calls and announcements, while users can discover and track available opportunities in real time.

---

## Features

- Secure Firebase Authentication
- Role-Based Access Control
- Funding Call Management
- Announcement System
- Real-Time Database Updates
- Admin Dashboard
- Analytics Dashboard
- Expired Funding Management
- Responsive UI
- Progressive Web App (PWA)
